## 🎯 部署步骤（超简单）

### 步骤 1：准备 GitHub 账号（2 分钟）

```
1. 访问 https://github.com
2. 点击"Sign up"注册
3. 填写邮箱、密码
4. 验证邮箱
5. ✅ 完成！
```

### 步骤 2：上传代码到 GitHub（3 分钟）

**在电脑上操作**：

```bash
# 1. 打开命令行
# 按 Win+R，输入 cmd，回车

# 2. 进入项目目录
cd d:\桂在安康

# 3. 初始化 Git
git init

# 4. 添加所有文件
git add .

# 5. 提交
git commit -m "我的健康系统"

# 6. 关联 GitHub 仓库（替换为你的用户名）
git remote add origin https://github.com/你的用户名/gui-zai-an-kang.git

# 7. 推送
git push -u origin main
```

**或者使用 GitHub Desktop（更简单）**：
```
1. 下载 GitHub Desktop
2. 登录 GitHub 账号
3. 把项目文件夹拖进去
4. 点击"Commit to main"
5. 点击"Publish repository"
```

### 步骤 3：部署到 Vercel（2 分钟）

```
1. 访问 https://vercel.com

2. 点击"Sign Up"，用 GitHub 账号登录

3. 点击"Add New Project"

4. 选择"Import Git Repository"

5. 找到你的项目 gui-zai-an-kang

6. 点击"Import"

7. 点击"Deploy"

8. 等待 1-2 分钟

9. ✅ 完成！你会看到一个网址：
   https://gui-zai-an-kang.vercel.app
```
