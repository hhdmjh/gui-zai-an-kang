## 🚀 部署步骤（只需 5 分钟）

### 第 1 步：注册 GitHub（2 分钟）

```
1. 打开浏览器：https://github.com
2. 点击"Sign up"
3. 填写：
   - 邮箱：您的邮箱
   - 密码：设置密码
   - 用户名：设置一个用户名
4. 完成人机验证
5. 打开邮箱，点击验证链接
6. ✅ GitHub 注册完成！
```

### 第 2 步：创建仓库（1 分钟）

```
1. 访问：https://github.com/new
2. Repository name: gui-zai-an-kang
3. 选择"Public"（公开）
4. 点击"Create repository"
5. ✅ 仓库创建完成！
```

### 第 3 步：推送代码到 GitHub（1 分钟）

**在 GitHub 仓库页面，复制这些命令**：

```bash
git remote add origin https://github.com/您的用户名/gui-zai-an-kang.git
git branch -M main
git push -u origin main
```

**然后在电脑上执行**：

```
1. 打开 PowerShell 或 CMD
2. 输入：cd d:\桂在安康
3. 粘贴上面的 3 行命令
4. 按回车
5. ✅ 代码推送成功！
```

### 第 4 步：部署到 Vercel（1 分钟）

```
1. 访问：https://vercel.com
2. 点击"Sign Up"
3. 选择"Continue with GitHub"
4. 完成授权
5. 点击"Add New Project"
6. 选择 gui-zai-an-kang 项目
7. 点击"Deploy"
8. 等待 30 秒
9. ✅ 部署完成！！！
```
