## 🎯 方案一：Railway 部署（推荐）

### 步骤 1: 安装 Node.js

1. 访问 https://nodejs.org
2. 下载并安装 LTS 版本（18.x 或更高）
3. 安装完成后，打开命令提示符，输入：
   ```bash
   node -v
   npm -v
   ```
   确认安装成功

### 步骤 2: 初始化 Git 仓库

在项目根目录（`d:\桂在安康`）执行：

```bash
git init
git add .
git commit -m "Initial commit"
```

### 步骤 3: 推送到 GitHub

1. 访问 https://github.com
2. 创建新仓库，名字：`gui-an-kang`
3. 不要勾选"Initialize this repository with a README"
4. 在命令提示符执行：
   ```bash
   git remote add origin https://github.com/你的用户名/gui-an-kang.git
   git push -u origin main
   ```

### 步骤 4: 部署到 Railway

1. 访问 https://railway.app
2. 使用 GitHub 账号登录
3. 点击 "New Project"
4. 选择 "Deploy from GitHub repo"
5. 选择 `gui-an-kang` 仓库
6. Railway 会自动识别并部署

### 步骤 5: 配置环境变量

在 Railway 项目页面：
1. 点击 "Variables" 标签
2. 添加以下变量：
   ```
   PORT=3000
   JWT_SECRET=gui-an-kang-secret-2024-change-this
   NODE_ENV=production
   ```

### 步骤 6: 配置 Root Directory

1. 点击 "Settings" 标签
2. 找到 "Root Directory"
3. 设置为：`backend`
4. 保存后会自动重新部署

### 步骤 7: 获取访问地址

部署成功后，Railway 会提供：
- 公网地址：`https://gui-an-kang-production.up.railway.app`

### 步骤 8: 修改前端 API 地址

修改 `api-config.js` 文件：

```javascript
const API_BASE_URL = 'https://你的项目名-production.up.railway.app/api';
```

然后提交推送到 GitHub：
```bash
git add .
git commit -m "Update API URL"
git push
```

### 步骤 9: 部署前端到 GitHub Pages

1. 访问 https://github.com/你的用户名/gui-an-kang/settings/pages
2. Source 选择 "Deploy from a branch"
3. Branch 选择 "main"，文件夹选择 "root"
4. 保存后等待部署完成
5. 获得地址：`https://你的用户名.github.io/gui-an-kang/`

## ❓ 常见问题

### Q: Railway 部署失败？
A: 
1. 检查 `backend/package.json` 是否存在
2. 确认 Root Directory 设置为 `backend`
3. 查看 Railway 的 Deploy Logs 找错误

### Q: 前端访问提示网络错误？
A:
1. 确认后端已部署成功
2. 检查 `api-config.js` 中的 API 地址是否正确
3. Railway 项目是否处于活跃状态（免费计划有休眠）

### Q: 数据库在哪里？
A: 
- 本地：`backend/gui-an-kang.db`
- Railway: 临时存储，重启会重置
- 持久化：需要 Railway 的数据库插件

### Q: 如何重置数据？
A:
1. 本地：删除 `backend/gui-an-kang.db`
2. Railway: 重新部署或手动删除数据库
