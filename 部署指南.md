# 🌿 桂在安康 - 完整部署指南

## 📋 项目简介

桂在安康是一个面向老年人的个人健康监测平台，具备以下核心功能:
- ✅ 健康数据记录与分析 (血压、血糖、睡眠、体重等)
- ✅ 智能健康预警系统
- ✅ 社区服务预约
- ✅ 健康知识库
- ✅ 用户认证与数据持久化
- ✅ 数据可视化图表

## 🚀 快速启动

### 第一步：安装 MySQL 数据库

1. 访问 https://dev.mysql.com/downloads/mysql/
2. 下载并安装 MySQL for Windows
3. 或使用 XAMPP (推荐初学者): https://www.apachefriends.org/

### 第二步：配置数据库连接

1. 编辑 `backend\.env` 文件:
```env
PORT=3000
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=你的 MySQL 密码
DB_NAME=gui_zai_an_kang
JWT_SECRET=gui_zai_an_kang_secret_key_2024
```

### 第三步：安装依赖

打开命令行，进入项目目录:
```bash
cd backend
npm install
```

### 第四步：初始化数据库

确保 MySQL 服务已启动，然后运行:
```bash
npm run init-db
```

看到以下提示表示成功:
```

🎉 数据库初始化完成！
```

### 第五步：启动后端服务

```bash
npm start
```

看到以下提示表示成功:

在浏览器中打开: `http://localhost:3000`

## 🔧 常见问题

### 1. MySQL 连接失败
- 检查 MySQL 服务是否启动
- 确认 `.env` 文件中的数据库配置正确
- 确保数据库 `gui_zai_an_kang` 已创建

### 2. 端口被占用
修改 `backend\.env` 中的 `PORT` 值，如 `PORT=3001`

### 3. npm install 失败
使用淘宝镜像:
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

## 📊 API 接口文档

### 认证相关
- `POST /api/register` - 用户注册
- `POST /api/login` - 用户登录
- `GET /api/user/me` - 获取当前用户信息
- `PUT /api/user/me` - 更新用户信息
- `POST /api/user/change-password` - 修改密码

### 健康数据
- `GET /api/health-data` - 获取健康数据
- `POST /api/health-data` - 保存健康数据
- `DELETE /api/health-data/:id` - 删除健康数据
- `GET /api/statistics` - 获取统计数据

### 预约服务
- `GET /api/appointments` - 获取预约记录
- `POST /api/appointments` - 创建预约
- `DELETE /api/appointments/:id` - 取消预约

### 预警系统
- `GET /api/warnings` - 获取预警记录
- `PUT /api/warnings/:id/read` - 标记为已读

### 社区功能
- `GET /api/events` - 获取活动列表
- `POST /api/events/:id/register` - 报名参加活动
- `GET /api/knowledge` - 获取健康知识
- `GET /api/knowledge/:id` - 获取知识详情
