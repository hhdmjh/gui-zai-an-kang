# 🚀 桂在安康 - 快速启动指南

## ✅ 后端已启动！

后端服务正在运行：
- 📡 API 地址：`http://localhost:3000/api`
- 💾 数据库：内存数据库（重启会重置）

## 🌐 立即使用

### 方式 1：直接打开前端

1. 打开浏览器
2. 访问：`file:///d:/桂在安康/login.html`
3. 注册新账号并登录
4. 开始使用所有功能

### 方式 2：通过后端访问前端

1. 打开浏览器
2. 访问：`http://localhost:3000/login.html`
3. 注册新账号并登录

## 📋 测试步骤

### 1. 注册账号
- 打开 `login.html`
- 点击"立即注册"
- 填写用户名和密码
- 完成注册

### 2. 登录系统
- 使用刚注册的账号登录
- 成功后会跳转到首页

### 3. 录入健康数据
- 在首页填写今日健康数据
- 点击"保存今日数据"
- 数据会保存到后端数据库

### 4. 查看 AI 报告
- 点击"AI 报告"
- 查看智能健康分析
- 获取个性化建议

### 5. 添加联系人
- 先注册第二个账号
- 用第一个账号登录
- 在联系人页面添加第二个账号的用户名
- 当健康数据异常时，系统会推送通知给联系人

## 🔧 重启后端

如果后端停止了，重新运行：

```bash
cd "d:\桂在安康\backend"
npm start
```

## 📊 数据库说明

当前使用**内存数据库**：
- ✅ 速度快
- ✅ 无需配置文件
- ❌ 重启后数据会丢失

**注意**：每次重启后端，所有数据会清空，需要重新注册账号和录入数据。

## 🌍 部署到公网（让别人访问）

### 步骤 1：推送到 GitHub

```bash
cd "d:\桂在安康"
git init
git add .
git commit -m "Initial commit"
```

然后在 GitHub 创建仓库并推送。

### 步骤 2：部署到 Railway

1. 访问 https://railway.app
2. 使用 GitHub 登录
3. 创建新项目，选择你的仓库
4. 设置 Root Directory 为 `backend`
5. 添加环境变量：
   - `PORT=3000`
   - `JWT_SECRET=your-secret-key`
6. 部署成功后获得公网地址

### 步骤 3：修改前端 API 地址

编辑 `api-config.js`：
```javascript
const API_BASE_URL = 'https://你的项目-production.up.railway.app/api';
```

### 步骤 4：部署前端到 GitHub Pages

1. GitHub 仓库 -> Settings -> Pages
2. 选择从 main 分支部署
3. 获得地址：`https://你的用户名.github.io/gui-zai-an-kang/`
