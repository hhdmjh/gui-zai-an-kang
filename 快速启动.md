# 🚀 桂在安康 - 后端快速启动指南

## 方法一：一键安装（推荐）

双击运行 `install-and-start.bat` 文件

## 方法二：手动安装

### 1️⃣ 安装 Node.js

**Windows 用户：**
```powershell
winget install OpenJS.NodeJS.LTS
```

**或手动下载：**
- 访问：https://nodejs.org/
- 下载 LTS 版本（长期支持版）
- 运行安装程序

**验证安装：**
```bash
node --version
npm --version
```

### 2️⃣ 安装 MySQL

**方式 A：MySQL 官方**
1. 访问：https://dev.mysql.com/downloads/mysql/
2. 下载 MySQL Installer for Windows
3. 安装并记住 root 密码

**方式 B：XAMPP（更简单）**
1. 访问：https://www.apachefriends.org/
2. 下载并安装 XAMPP
3. 启动 XAMPP Control Panel
4. 点击 MySQL 的 Start 按钮

### 3️⃣ 配置项目

打开命令提示符，进入项目目录：
```bash
cd "d:\桂在安康\backend"
```

复制环境配置文件：
```bash
copy .env.example .env
```

编辑 `.env` 文件，修改数据库配置：
```env
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=你的 MySQL 密码
DB_NAME=gui_zai_an_kang
```

### 4️⃣ 安装依赖

```bash
npm install
```

### 5️⃣ 初始化数据库

确保 MySQL 已启动，然后运行：
```bash
npm run init-db
```

### 6️⃣ 启动后端服务

```bash
npm start
```

看到以下信息表示成功：

## ✅ 验证安装

打开浏览器访问：
- http://localhost:3000 - 查看服务状态
- http://localhost:3000/api/health - 检查数据库连接

## 🔧 前端集成

在 HTML 文件中添加：
```html
<script src="backend/api-client.js"></script>
```

然后使用 `SyncManager` 替代 `localStorage`：
```javascript
// 保存数据
await SyncManager.saveHealthData(data);

// 获取数据
let healthData = await SyncManager.getHealthData();
```

## ❓ 常见问题

### Q: Node.js 安装失败？
A: 以管理员身份运行安装程序，或重启电脑后重试。

### Q: MySQL 连接失败？
A: 检查 MySQL 服务是否启动，确认密码正确。

### Q: 端口 3000 被占用？
A: 修改 `.env` 文件中的 `PORT=3001`

### Q: npm install 很慢？
A: 使用淘宝镜像：
```bash
npm config set registry https://registry.npmmirror.com
npm install
```
