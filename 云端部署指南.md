# 🚀 桂在安康 - 云端部署方案

## 📋 部署目标
让所有人都能随时随地通过互联网访问系统，像 APP 一样方便！

---

## 🌐 方案一：使用免费云平台（推荐）

### 1. Vercel 部署（最简单，免费）

**步骤**：

#### 1.1 准备 GitHub 账号
```
1. 访问 https://github.com
2. 注册账号（如果没有）
3. 创建新仓库：gui-zai-an-kang
```

#### 1.2 上传代码到 GitHub
```bash
# 在项目根目录执行
cd d:\桂在安康

# 初始化 Git
git init

# 添加所有文件
git add .

# 提交
git commit -m "初始版本"

# 关联 GitHub 仓库
git remote add origin https://github.com/你的用户名/gui-zai-an-kang.git

# 推送
git push -u origin main
```

#### 1.3 部署到 Vercel
```
1. 访问 https://vercel.com
2. 用 GitHub 账号登录
3. 点击"New Project"
4. 导入刚才的仓库
5. 点击"Deploy"
6. 等待部署完成
```

#### 1.4 获取访问地址
```
部署完成后，Vercel 会给你一个网址：
https://gui-zai-an-kang.vercel.app

任何人都可以访问这个网址使用系统！
```

---

### 2. Railway 部署（带数据库，免费）

**步骤**：

#### 2.1 注册 Railway
```
1. 访问 https://railway.app
2. 用 GitHub 账号登录
3. 点击"New Project"
```

#### 2.2 创建 MySQL 数据库
```
1. 点击"New" → "Database" → "MySQL"
2. 等待数据库创建
3. 记录连接信息：
   - DATABASE_URL
   - 主机名
   - 用户名
   - 密码
```

#### 2.3 部署后端
```
1. 点击"New" → "GitHub Repo"
2. 选择你的项目
3. 添加环境变量：
   - DATABASE_URL=你的数据库连接
4. 点击"Deploy"
```

#### 2.4 获取访问地址
```
部署完成后，Railway 会给你：
https://你的项目.up.railway.app

所有人都可以访问！
```

## 🌐 方案二：国内云平台（速度快）

### 1. 腾讯云云开发（推荐）

**步骤**：

#### 1.1 注册腾讯云
```
1. 访问 https://cloud.tencent.com
2. 注册账号
3. 实名认证（必须）
```

#### 1.2 开通云开发
```
1. 搜索"云开发 CloudBase"
2. 点击"开通服务"
3. 选择"免费版"
4. 创建环境
```

#### 1.3 部署项目
```
1. 下载 CloudBase CLI
   npm install -g @cloudbase/cli

2. 登录
   tcb login

3. 部署
   tcb framework deploy

4. 获取访问地址
   https://你的环境-id.service.tcloudbase.com
```

---

## 🌐 方案三：自己搭建服务器（长期稳定）

### 购买云服务器

#### 1. 选择服务商
```
- 腾讯云：https://cloud.tencent.com
- 阿里云：https://aliyun.com
- 华为云：https://huawei.com
```

#### 2. 选择配置（最低配置即可）
```
- CPU: 1 核
- 内存：2GB
- 硬盘：40GB
- 带宽：3Mbps
- 价格：约 60-100 元/月
```

#### 3. 安装系统
```
选择：Ubuntu 20.04 或 CentOS 7
```

#### 4. 连接服务器
```bash
# Windows 使用 PowerShell
ssh root@你的服务器 IP

# 输入密码登录
```

#### 5. 安装环境
```bash
# 安装 Node.js
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# 安装 MySQL
sudo apt-get install mysql-server

# 安装 Nginx
sudo apt-get install nginx
```

#### 6. 上传代码
```bash
# 使用 Git
git clone https://github.com/你的用户名/gui-zai-an-kang.git

# 或使用 FTP 工具上传
```

#### 7. 启动服务
```bash
cd gui-zai-an-kang/backend
npm install
node demo-server.js
```

#### 8. 配置域名（可选）
```
1. 购买域名（约 50 元/年）
2. 解析到服务器 IP
3. 配置 Nginx 反向代理
```

## 📱 创建手机 APP（可选）

### 使用 PWA 技术

#### 1. 添加 manifest.json
```json
{
  "name": "桂在安康",
  "short_name": "桂在安康",
  "start_url": "/login.html",
  "display": "standalone",
  "background_color": "#ffffff",
  "theme_color": "#2d8a5f",
  "icons": [
    {
      "src": "/icon-192.png",
      "sizes": "192x192",
      "type": "image/png"
    }
  ]
}
```

#### 2. 在 HTML 中添加
```html
<link rel="manifest" href="/manifest.json">
<meta name="theme-color" content="#2d8a5f">
<meta name="apple-mobile-web-app-capable" content="yes">
```

#### 3. 用户可以添加到桌面
```
手机浏览器打开网址 → 点击菜单 → 添加到主屏幕
```
