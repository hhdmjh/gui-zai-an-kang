### 3️⃣ 数据库内容

**7 张表：**

1. **users** - 用户表
   - 已有数据：admin、test 两个账号
   - 字段：用户名、密码（加密）、邮箱、手机、真实姓名等

2. **health_data** - 健康数据表
   - 字段：血压、血糖、睡眠、体重、运动等
   - 支持：按日期查询、统计分析

3. **appointments** - 预约记录表
   - 字段：预约时间、地址、服务类型、状态

4. **warnings** - 预警记录表
   - 字段：预警类型、级别、消息、是否已读

5. **knowledge_articles** - 知识库表
   - 字段：文章标题、内容、分类、浏览量

6. **community_events** - 社区活动表
   - 字段：活动信息、时间、地点、报名人数

7. **event_registrations** - 活动报名表
   - 字段：用户 ID、活动 ID、报名状态

---

### 4️⃣ API 接口（17 个）

**用户认证（5 个）：**
- POST /api/register - 用户注册
- POST /api/login - 用户登录
- GET /api/user/me - 获取当前用户
- PUT /api/user/me - 更新用户信息
- POST /api/user/change-password - 修改密码

**健康数据（4 个）：**
- GET /api/health-data - 获取健康数据
- POST /api/health-data - 保存健康数据
- DELETE /api/health-data/:id - 删除健康数据
- GET /api/statistics - 获取统计数据

**预警系统（2 个）：**
- GET /api/warnings - 获取预警记录
- PUT /api/warnings/:id/read - 标记已读

**预约服务（3 个）：**
- GET /api/appointments - 获取预约
- POST /api/appointments - 创建预约
- DELETE /api/appointments/:id - 取消预约

**社区功能（3 个）：**
- GET /api/events - 获取活动列表
- GET /api/knowledge - 获取知识列表
- GET /api/knowledge/:id - 获取知识详情

---

### 5️⃣ 前端页面（8 个）
