## 📊 数据库表结构

### users - 用户表
```
- id: 用户 ID
- username: 用户名
- password: 加密密码
- real_name: 真实姓名
- phone: 手机号
- email: 邮箱
- gender: 性别
- age: 年龄
- role: 角色
- created_at: 注册时间
```

### health_data - 健康数据表
```
- id: 记录 ID
- user_id: 用户 ID
- record_date: 记录日期
- systolic: 收缩压
- diastolic: 舒张压
- blood_sugar: 血糖
- sleep: 睡眠
- weight: 体重
- sport: 运动
- created_at: 创建时间
```

### ai_health_reports - AI 报告表
```
- id: 报告 ID
- user_id: 用户 ID
- report_date: 报告日期
- overall_score: 综合评分
- bp_score: 血压评分
- sugar_score: 血糖评分
- lifestyle_score: 生活方式评分
- suggestions: 健康建议
- warning_suggestions: 预警建议
- trend_analysis: 趋势分析
- created_at: 创建时间
```
