# 桂在安康 - 作品简介

## 📋 作品简介（100 字内）

**桂在安康**是一款面向社区的健康智能管理系统，融合广西特色医药文化。系统提供健康数据监测、AI 健康分析、智能预警、健康知识库等功能，采用前后端分离架构，支持云端部署。帮助用户科学管理健康，预防疾病，提升生活质量，助力健康中国战略。

---

## 🎯 核心亮点

1. **智能健康监测** - 血压、血糖、睡眠等多项指标实时追踪
2. **AI 健康分析** - 基于数据生成个性化健康建议
3. **智能预警系统** - 异常数据自动预警，推送家属和医生
4. **广西特色知识库** - 85 篇健康知识，融合药食同源智慧
5. **云端部署架构** - 前后端分离，支持全球访问

---

## 💡 创新点

- 结合广西地域特色的健康管理方案
- 采用 Node.js + Express 后端，MySQL 数据库
- JWT 身份验证，保障数据安全
- ECharts 数据可视化，直观展示健康趋势
- 响应式设计，适配多终端

---

## 🏆 适用场景

- 社区健康服务中心
- 养老机构健康管理
- 个人日常健康监测
- 慢性病患者的长期跟踪

---

**作品名称**：桂在安康 - 广西社区健康智能检测与预警系统

**技术栈**：HTML5 + CSS3 + JavaScript + Node.js + Express + MySQL + ECharts

**部署方式**：Vercel（前端）+ Railway（后端）
