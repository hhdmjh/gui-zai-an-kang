**在 GitHub 仓库页面往下拉，看到"…or push an existing repository from the command line"，复制下面 3 行命令**：

```bash
git remote add origin https://github.com/您的用户名/gui-zai-an-kang.git
git branch -M main
git push -u origin main
```

**然后在电脑上打开 CMD 或 PowerShell，运行这 3 行命令**
