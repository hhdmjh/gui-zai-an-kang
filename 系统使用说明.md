# 🌿 桂在安康 - 系统使用说明

## 📱 别人怎么用我的系统？

### 方式一：本地部署（推荐用于演示）
1. **准备环境**：
   - 确保已安装 Node.js（运行 `node -v` 检查）
   - 确保已安装 MySQL（运行 `mysql -V` 检查）

2. **启动系统**：
   ```bash
   # 方式 1：一键启动（推荐）
   双击"启动检查.bat"
   
   # 方式 2：手动启动
   cd backend
   npm install
   node server.js
   ```

3. **访问系统**：
   - 打开浏览器访问：`http://localhost:3000`
   - 或直接打开 `login.html` 文件

4. **登录账号**：
   - 演示账号：`admin` / `123456`
   - 或点击"立即注册"创建新账号

## 🔄 日期记录重置功能

### 位置
打开 `home.html` → 页面顶部"数据管理"区域

### 三个功能按钮

#### 1️⃣ 重置今日数据
- **功能**：删除今天的健康数据记录
- **适用场景**：今天的数据填错了，需要重新填写
- **操作**：点击按钮 → 确认 → 完成

#### 2️⃣ 清空所有数据
- **功能**：删除所有历史健康数据
- **适用场景**：重新开始记录，或清除测试数据
- **警告**：⚠️ 此操作不可恢复，请谨慎使用
- **操作**：点击按钮 → 仔细阅读警告 → 确认 → 完成

#### 3️⃣ 导出数据
- **功能**：将所有健康数据导出为 Excel 可打开的 CSV 文件
- **适用场景**：备份数据、打印报告、给医生查看
- **格式**：CSV 文件（可用 Excel 打开）
- **内容**：日期、收缩压、舒张压、血糖、睡眠、体重、运动
