## 📱 让别人都能用

### 局域网访问：

**步骤：**

1. **确保所有电脑在同一网络**
   - 连接同一个 WiFi
   - 或同一个路由器

2. **查看你的 IP 地址**
   ```bash
   ipconfig
   ```

3. **其他人访问**
   ```
   http://你的 IP:3000
   ```

4. **如果访问不了**
   - 检查防火墙
   - 开放 3000 端口
