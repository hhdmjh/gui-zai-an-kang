## 方式二：创建桌面快捷方式（安装包效果）

### 步骤 1：创建启动脚本

在桌面新建文本文件，命名为 `启动桂在安康.txt`，内容：

```batch
@echo off
start https://hhdmjh.github.io/gui-zai-an-kang/
exit
```

### 步骤 2：转换为快捷方式
1. 右键 `启动桂在安康.txt`
2. 选择"重命名"，改为 `启动桂在安康.bat`
3. 双击即可启动

### 步骤 3：创建桌面图标（可选）
1. 右键 `启动桂在安康.bat`
2. 选择"发送到" -> "桌面快捷方式"
3. 右键快捷方式 -> "属性" -> "更改图标"
4. 选择一个喜欢的图标

## 方式四：本地安装包（离线使用）

### 创建安装包
1. 复制整个 `桂在安康` 文件夹
2. 压缩成 ZIP 文件：`桂在安康 v1.0.zip`
3. 分享给他人

### 使用方法
1. 解压 ZIP 文件到任意位置
2. 双击 `本地测试.bat` 启动
3. 浏览器自动打开，即可使用

### 优点
- ✅ 完全离线使用
- ✅ 不依赖网络
- ✅ 数据本地存储，更安全

### 缺点
- ❌ 需要电脑环境
- ❌ 手机无法使用

## 常见问题

### Q: 别人访问说打不开？
A: 
1. 检查网络连接
2. 清除浏览器缓存（Ctrl+Shift+Delete）
3. 强制刷新（Ctrl+Shift+R）
4. 尝试备用网址

### Q: 数据会同步吗？
A: 
- 目前数据存储在本地浏览器
- 不同设备数据不共享
- 这是设计特性，保护隐私

### Q: 可以离线使用吗？
A: 
- GitHub Pages 需要网络
- 如需离线使用，请下载完整文件包
- 运行 `本地测试.bat` 即可

### Q: 手机能用吗？
A: 
- 完全可以！
- 扫码访问即可
- 建议添加到主屏幕，体验更佳
