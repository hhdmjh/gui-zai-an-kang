## ✅ 第二步：配置环境（10 分钟）

### 1. 确认已安装 Node.js
打开命令行，输入：
```bash
node --version
```
应该显示版本号，如 `v18.16.0`

如果没安装，去 https://nodejs.org/ 下载安装

### 2. 确认已安装 MySQL
打开命令行，输入：
```bash
mysql --version
```
应该显示版本号

**如果没有 MySQL，选择以下任一方案：**
- 方案 A：安装 MySQL（https://dev.mysql.com/downloads/）
- 方案 B：安装 XAMPP（更简单，https://www.apachefriends.org/）

### 3. 配置数据库连接
用文本编辑器打开 `backend\.env` 文件：
```env
PORT=3000
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=这里填你的 MySQL 密码
DB_NAME=gui_zai_an_kang
JWT_SECRET=gui_zai_an_kang_secret_key_2024
```

**重要：** 必须修改 `DB_PASSWORD` 为你的 MySQL 密码！

## ❓ 常见问题解决

### Q1: npm install 失败
**解决：**
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

### Q2: MySQL 连接失败
**检查：**
1. MySQL 服务是否启动
2. `.env` 文件中的密码是否正确
3. 数据库 `gui_zai_an_kang` 是否创建

### Q3: 端口被占用
**解决：** 修改 `backend\.env` 中的端口
```env
PORT=3001
```
然后访问 `http://localhost:3001`

### Q4: 页面空白或报错
**检查：**
1. 服务器是否启动
2. 浏览器控制台是否有错误（F12 查看）
3. 清除浏览器缓存

### Q5: AI 报告页面找不到
**解决：** 直接在地址栏输入
```
http://localhost:3000/ai-report.html
```
