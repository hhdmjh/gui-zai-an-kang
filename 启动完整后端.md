## 🔧 步骤 1：启动 MySQL 服务

### 方法 A：使用 Windows 服务（推荐）

**1. 打开服务管理器**
- 按 `Win + R`
- 输入 `services.msc`
- 回车

**2. 找到 MySQL 服务**
- 在服务列表中找 "MySQL" 或 "MySQL80"
- 右键 → 启动

**3. 如果找不到服务**
说明 MySQL 没有注册为 Windows 服务，需要用方法 B。

---

### 方法 B：手动启动 MySQL

**1. 以管理员身份打开命令行**
- 按 `Win + X`
- 选择"Windows PowerShell（管理员）"或"命令提示符（管理员）"

**2. 进入 MySQL 的 bin 目录**
```bash
cd "D:\Program Files\MySQL8.0\bin"
```

**3. 启动 MySQL**
```bash
mysqld --console
```

**注意：** 这个窗口不要关闭！

---

## 🔑 步骤 2：找到或重置 MySQL 密码

### 尝试默认密码

MySQL 8.0 初始安装时会生成一个临时密码。

**1. 查看临时密码日志**
```bash
# 在命令行执行
type "C:\ProgramData\MySQL\MySQL Server 8.0\Data\*.err" | findstr "password"
```

**2. 常见密码位置**
- 安装时的截图或笔记
- 安装向导中显示的密码
- 可能保存在：`C:\ProgramData\MySQL\MySQL Server 8.0\Data\` 的 .err 文件中

---

### 如果找不到密码：重置为 123456

**1. 停止 MySQL（如果正在运行）**
```bash
# 在服务管理器中停止
# 或者在任务管理器中结束 mysqld.exe 进程
```

**2. 创建重置密码的 SQL 文件**

用记事本创建文件：`C:\mysql-reset.txt`
内容：
```sql
ALTER USER 'root'@'localhost' IDENTIFIED BY '123456';
FLUSH PRIVILEGES;
```

**3. 以跳过权限表模式启动**
```bash
cd "D:\Program Files\MySQL8.0\bin"
mysqld --init-file=C:\mysql-reset.txt --console
```

**4. 测试新密码**
打开新的命令行窗口：
```bash
cd "D:\Program Files\MySQL8.0\bin"
mysql -u root -p123456
```

如果看到 `mysql>` 提示符，说明密码重置成功！

输入 `EXIT;` 退出。

**5. 正常启动 MySQL**
```bash
cd "D:\Program Files\MySQL8.0\bin"
mysqld --console
```

## 💡 常见问题

### Q1: MySQL 服务启动失败？
**A:** 
- 检查端口 3306 是否被占用
- 查看错误日志：`C:\ProgramData\MySQL\MySQL Server 8.0\Data\*.err`
- 重启电脑后再试

### Q2: npm start 失败？
**A:**
- 确保 MySQL 正在运行
- 确保 .env 文件中的密码正确
- 确保已运行 `npm install`

### Q3: 登录失败？
**A:**
- 确保后端已启动
- 确保浏览器访问的是 http://localhost:3000
- 检查浏览器控制台是否有错误

### Q4: 数据库连接失败？
**A:**
- 检查 MySQL 是否运行
- 检查 .env 文件中的密码是否正确
- 运行 `npm run init-db` 重新初始化
