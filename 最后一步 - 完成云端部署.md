## 🚀 超简单步骤：

### 第 1 步：注册 GitHub（2 分钟）

```
1. 打开浏览器访问：https://github.com
2. 点击右上角"Sign up"
3. 填写：
   - 邮箱：输入您的邮箱
   - 密码：设置密码
   - 用户名：设置用户名（如：zhangsan）
4. 完成验证
5. 检查邮箱，输入验证码
6. ✅ 完成！
```

### 第 2 步：创建仓库（1 分钟）

```
1. 访问：https://github.com/new
2. Repository name: gui-zai-an-kang
3. 选择"Public"（公开）
4. 点击"Create repository"
5. ✅ 完成！
```

### 第 3 步：推送代码到 GitHub（1 分钟）

**在 GitHub 仓库页面，复制这行命令**（替换您的用户名）：
```bash
git remote add origin https://github.com/你的用户名/gui-zai-an-kang.git
```

**在 PowerShell 中执行**：
```
1. 打开 PowerShell（按 Win+X，选择 PowerShell）
2. 输入：cd d:\桂在安康
3. 粘贴：git remote add origin https://github.com/你的用户名/gui-zai-an-kang.git
4. 按回车
5. 输入：git push -u origin main
6. 按回车
7. ✅ 完成！
```

### 第 4 步：部署到 Vercel（1 分钟）

```
1. 访问：https://vercel.com
2. 点击"Sign Up"
3. 选择"Continue with GitHub"
4. 授权 Vercel 访问 GitHub
5. 点击"Add New Project"
6. 选择"Import Git Repository"
7. 找到 gui-zai-an-kang 项目
8. 点击"Import"
9. 点击"Deploy"
10. 等待 30 秒
11. ✅ 完成！
```
