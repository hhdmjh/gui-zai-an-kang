# 桂在安康 - 个人健康监测平台

## 项目简介

「桂在安康」是一个面向老年用户的个人健康监测平台，集成了健康数据管理、智能预警、社区服务对接和健康知识库四大核心功能模块。平台采用大字体、高对比度的老年友好设计，特别融入了广西地域特色的健康知识内容。

## 项目结构

```
桂在安康/
├── index.html          # 入口页面（欢迎页）
├── loogin.html         # 登录页
├── home.html           # 健康数据管理主页
├── community.html      # 社区服务页
├── knowledge.html      # 健康知识库页
├── style.css           # 统一样式文件
├── script.js           # 通用工具函数库
└── README.md           # 项目说明文档
```

## 核心功能模块

### 1. 健康数据管理模块（home.html）

**功能特点：**
- ✅ 数据录入表单：支持血压、血糖、睡眠、体重、运动等多项健康指标
- ✅ 本地存储：使用 localStorage 保存数据，刷新页面不丢失
- ✅ 趋势图表：使用 ECharts 生成可视化趋势图
- ✅ 智能预警：自动检测数据是否超标，显示预警信息
- ✅ 数据预测：基于线性回归算法预测健康指标趋势
- ✅ 预警推送：模拟向家属和医生发送预警通知

**预警阈值：**
- 收缩压：≥ 140 mmHg
- 舒张压：≥ 90 mmHg
- 血糖：≥ 7.0 mmol/L
- 睡眠：< 6 小时
- 体重：≥ 110 斤

### 2. 智能预警系统模块

**功能特点：**
- ✅ 实时检测：每次提交数据和页面加载时自动检测
- ✅ 阈值判断：符合老年健康标准的预警阈值
- ✅ 预警展示：超标数据自动显示，无异常自动隐藏
- ✅ 推送功能：模拟向家属、社区医生发送提醒

### 3. 社区服务对接模块（community.html）

**功能特点：**
- ✅ 上门体检预约：在线填写预约表单，选择时间
- ✅ 健康咨询：模拟对接社区医生咨询通道
- ✅ 机构查询：展示广西本地养老服务机构信息
- ✅ 预约记录：查看历史预约记录
- ✅ 导航功能：提供机构导航（模拟）

**服务内容包括：**
- 上门体检预约
- 健康咨询服务
- 养老机构查询
- 预约记录管理

### 4. 健康知识库模块（knowledge.html）

**功能特点：**
- ✅ 疾病分类：高血压、糖尿病、风湿、心脑血管等
- ✅ 广西特色：融入广西地域特色健康知识
- ✅ 卡片布局：每个疾病对应一个卡片，点击查看详情
- ✅ 药食同源：广西本地食疗方、养生汤谱

**广西特色内容：**
- 针对南方湿热气候的风湿护理知识
- 广西本地食疗方（薏米赤小豆汤、五指毛桃煲鸡等）
- 广西特色药材介绍（田七、罗汉果、肉桂等）
- 四季养生汤谱

## 使用说明

### 登录方式

1. 打开 `index.html` 进入欢迎页
2. 点击"进入平台"按钮
3. 使用演示账号登录：
   - 用户名：`admin`
   - 密码：`123456`

### 数据录入

1. 登录成功后进入健康数据管理页
2. 填写健康数据表单（血压、血糖、睡眠、体重、运动等）
3. 点击"提交数据"按钮
4. 数据自动保存到本地，并生成趋势图表

### 查看预警

- 如果提交的数据超过预警阈值，预警栏会自动显示
- 点击"发送预警通知"可模拟发送通知给家属和医生

### 社区服务

1. 点击导航栏"社区服务"进入服务页
2. 可选择上门体检预约、健康咨询、机构查询等服务
3. 预约记录会自动保存并显示

### 健康知识

1. 点击导航栏"健康知识库"进入知识库
2. 按疾病分类查看健康知识
3. 点击卡片查看详细内容
4. 特别推荐"广西特色"板块

1. 本系统为演示版本，数据存储在本地浏览器中

## 开发者说明

### 工具函数库（script.js）

提供了以下通用工具：

- `StorageUtils`：本地存储管理
- `UserUtils`：用户管理
- `DateUtils`：日期处理
- `NotificationUtils`：通知提示
- `HealthDataUtils`：健康数据管理
- `WarningUtils`：预警判断
- `AppointmentUtils`：预约管理
- `PredictionUtils`：预测算法
- `ChartUtils`：图表管理
- `FormValidateUtils`：表单验证

### 样式变量（style.css）

CSS 变量定义在 `:root` 中，方便统一修改：

```css
--primary-color: #2d8a5f;      /* 主色调 */
--font-base: 18px;             /* 基础字体 */
--font-large: 22px;            /* 大字体 */
```
