# 🌿 桂在安康 - 作品完整总结

## 📋 项目概述

**项目名称**：桂在安康 - 广西社区健康智能检测与预警系统

**项目定位**：融合广西特色医药文化的智能化健康管理平台

**技术架构**：前后端分离的 Web 应用系统

---

## 📁 项目结构

### 一、前端部分（8 个 HTML 页面）

#### 1. 核心功能页面
- **index.html** - 系统首页/引导页
- **login.html** - 用户登录页面
- **register.html** - 用户注册页面
- **home.html** - 健康数据管理主页（核心功能）
- **profile.html** - 个人中心页面

#### 2. 特色功能页面
- **contacts.html** - 紧急联系人管理 ⭐
- **notifications.html** - 消息中心/预警接收 ⭐
- **ai-report.html** - AI 健康分析报告 ⭐

#### 3. 服务页面
- **community.html** - 社区服务页面
- **knowledge.html** - 健康知识库（85 篇文章）

#### 4. 公共资源
- **style.css** - 全局样式文件
- **script.js** - 公共 JavaScript 脚本

---

### 二、后端部分（Node.js + Express）

#### 1. 核心服务器
- **backend/full-server.js** - 主服务器（完整功能）

  - AI 健康分析
  - 联系人管理
  - 消息推送系统

#### 2. 数据库模块
- **backend/database/db.js** - 数据库配置

- **backend/database/demo-db.js** - 演示数据库
- **backend/database/clear-all-data.sql** - 数据清理脚本

#### 3. 中间件
- **backend/middleware/auth.js** - JWT 身份认证中间件

#### 4. API 路由
- **backend/routes/api.js** - RESTful API 接口（17+ 个端点）

#### 5. 配置文件
- **backend/package.json** - Node.js 依赖配置
- **backend/.env** - 环境变量配置
- **backend/Procfile** - Railway 部署配置

## 💻 技术栈

### 前端技术
- **HTML5** - 页面结构
- **CSS3** - 样式设计（响应式布局）
- **JavaScript (ES6+)** - 交互逻辑
- **ECharts** - 数据可视化图表
- **LocalStorage** - 本地数据存储

### 后端技术
- **Node.js v18+** - 运行环境
- **Express 4.x** - Web 框架
- **JWT** - 身份认证
- **bcryptjs** - 密码加密
- **CORS** - 跨域支持

### 数据库
- **MySQL 8.0** - 关系型数据库
- **8 张数据表**：
  - users（用户表）
  - health_data（健康数据表）
  - ai_health_reports（AI 报告表）
  - appointments（预约表）
  - warnings（预警表）
  - knowledge_articles（知识文章表）
  - community_events（社区活动表）
  - event_registrations（活动报名表）

### 部署平台
- **Vercel** - 前端托管（CDN 加速）
- **Railway** - 后端云服务
- **GitHub** - 代码仓库
