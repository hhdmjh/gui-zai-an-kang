### 3. Railway（后端）- 需要手动操作

#### 快速步骤：
1. **访问 Railway**
   - 打开：https://railway.app
   - 点击 "Login" → "Login with GitHub"
   - 使用 GitHub 账号登录

2. **创建项目**
   - 点击 "New Project"
   - 选择 "Deploy from GitHub repo"
   - 选择 `gui-zai-an-kang`

3. **配置设置**
   - 点击项目 → "Services" → "gui-zai-an-kang"
   - 点击 "Settings"
   - 设置 **Root Directory**: `backend`
   - 添加环境变量（Variables）:
     ```
     PORT = 3000
     JWT_SECRET = gui-an-kang-secret-2024
     NODE_ENV = production
     ```

4. **等待部署**
   - Railway 会自动构建和部署
   - 成功后会显示 "Deployed"
   - 获得公网地址：`https://gui-zai-an-kang-production.up.railway.app`
