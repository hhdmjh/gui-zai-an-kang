### 如果你知道 MySQL 密码

打开文件：`D:\桂在安康\backend\.env`

```env

```

**2. 双击运行：**
```

```

**3. 等待安装完成**

**4. 双击启动：**
```
D:\桂在安康\start.bat
```

**5. 访问系统：**
```

登录：admin / 123456
```
