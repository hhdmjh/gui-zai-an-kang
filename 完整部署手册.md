# 🌿 桂在安康 - 完整部署手册（生产环境）

## 📋 系统要求

### 服务器配置
- **操作系统**: Windows 10/11 或 Linux
- **内存**: 至少 4GB
- **硬盘**: 至少 10GB 可用空间
- **网络**: 需要开放端口 3000

### 必需软件
1. **Node.js 18+** - https://nodejs.org/
2. **MySQL 8.0+** - https://dev.mysql.com/downloads/

---

## 🚀 快速部署步骤

### 第 1 步：安装 MySQL 8.0

**Windows 安装：**

1. 下载 MySQL Installer
   - 访问：https://dev.mysql.com/downloads/installer/
   - 下载 `mysql-installer-community-8.0.xx.msi`

2. 运行安装程序
   - 选择 "Developer Default" 或 "Server only"
   - 设置 root 密码：**记住这个密码！**
   - 其他选项保持默认

3. 验证安装
   ```bash
   mysql --version
   ```

**重要：安装时设置的 root 密码一定要记住！**

---

### 第 2 步：安装 Node.js

**Windows 安装：**

1. 下载 Node.js LTS
   - 访问：https://nodejs.org/
   - 下载 "LTS" 版本（推荐 v18 或 v20）

2. 运行安装程序
   - 一直点击"下一步"
   - 使用默认设置

3. 验证安装
   ```bash
   node --version
   npm --version
   ```

---

### 第 3 步：部署项目

**1. 复制项目文件**

将整个 `桂在安康` 文件夹复制到目标电脑：
```
D:\桂在安康\
```

**2. 配置数据库连接**

编辑文件：`backend\.env`

```env
# 服务器配置
PORT=3000

# MySQL 数据库配置
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=你的 MySQL 密码
DB_NAME=gui_zai_an_kang

# JWT 密钥（可以随机生成）
JWT_SECRET=gui_zai_an_kang_secret_key_2024
```

**关键：修改 `DB_PASSWORD` 为你的 MySQL 密码**

---

### 第 4 步：安装依赖

打开命令行，执行：
```bash
cd D:\桂在安康\backend
npm install
```

等待安装完成（大约 2-5 分钟）

## 🌐 让别人都能用的方法

### 方案 1：局域网部署（推荐）

**适用场景：** 学校机房、办公室、家庭网络

**步骤：**

1. 在一台电脑上启动后端（作为服务器）
2. 确保所有电脑在同一局域网
3. 其他电脑通过 IP 地址访问

**优点：**
- ✅ 速度快
- ✅ 不需要公网 IP
- ✅ 安全

**缺点：**
- ❌ 只能在同一网络使用

---

### 方案 2：云服务器部署

**适用场景：** 互联网访问、多个地点使用

**推荐云服务商：**
- 阿里云：https://www.aliyun.com/
- 腾讯云：https://cloud.tencent.com/
- 华为云：https://www.huaweicloud.com/

**配置要求（最低）：**
- CPU: 1 核
- 内存：2GB
- 硬盘：40GB
- 带宽：1Mbps

**部署步骤：**

1. 购买云服务器（选择 CentOS 或 Ubuntu）
2. 安装 Node.js 和 MySQL
3. 上传项目代码
4. 配置 .env 文件
5. 启动服务
6. 开放端口 3000

**其他人访问：**
```
http://云服务器公网IP:3000
```

**优点：**
- ✅ 任何地方都能访问
- ✅ 24 小时运行
- ✅ 专业可靠

**缺点：**
- ❌ 需要付费（约 100-300 元/年）

---

### 方案 3：内网穿透（免费方案）

**适用场景：** 临时演示、测试

**工具推荐：**
- ngrok: https://ngrok.com/
- natapp: https://natapp.cn/

**使用 ngrok：**

1. 下载 ngrok
2. 注册获取 token
3. 启动后端后，运行：
   ```bash
   ngrok http 3000
   ```
4. 获得一个公网 URL，例如：
   ```
   https://abc123.ngrok.io
   ```

**优点：**
- ✅ 免费
- ✅ 立即获得公网地址

**缺点：**
- ❌ 每次启动 URL 都不同
- ❌ 速度较慢
- ❌ 不稳定

## 📦 制作安装包（让别人一键安装）

### 创建安装脚本

**1. 创建 install.bat（Windows）**

文件位置：`D:\桂在安康\install.bat`

内容：
```batch
@echo off
chcp 65001 >nul
echo ╔════════════════════════════════════════════════════════╗
echo ║          🌿 桂在安康 - 一键安装程序                   ║
echo ╚════════════════════════════════════════════════════════╝
echo.

REM 检查 Node.js
node --version >nul 2>&1
if %errorlevel% neq 0 (
    echo ❌ 未安装 Node.js
    echo 请先安装 Node.js: https://nodejs.org/
    pause
    exit /b 1
)
echo ✅ Node.js 已安装

REM 检查 MySQL
mysql --version >nul 2>&1
if %errorlevel% neq 0 (
    echo ❌ 未安装 MySQL
    echo 请先安装 MySQL: https://dev.mysql.com/downloads/
    pause
    exit /b 1
)
echo ✅ MySQL 已安装

REM 进入 backend 目录
cd backend
if %errorlevel% neq 0 (
    echo ❌ 无法进入 backend 目录
    pause
    exit /b 1
)

REM 复制环境配置
if not exist .env (
    echo 正在创建配置文件...
    copy .env.example .env
    echo.
    echo ⚠️  请编辑 backend\.env 文件
    echo 修改 DB_PASSWORD 为你的 MySQL 密码
    echo.
    pause
)

REM 安装依赖
echo 正在安装依赖...
call npm install
if %errorlevel% neq 0 (
    echo ❌ 依赖安装失败
    pause
    exit /b 1
)
echo ✅ 依赖安装成功

REM 初始化数据库
echo 正在初始化数据库...
call npm run init-db
if %errorlevel% neq 0 (
    echo ❌ 数据库初始化失败
    echo 请检查 MySQL 是否运行，以及 .env 文件中的密码是否正确
    pause
    exit /b 1
)
echo ✅ 数据库初始化成功

echo.
echo ╔════════════════════════════════════════════════════════╗
echo ║              安装完成！                                ║
echo ╠════════════════════════════════════════════════════════╣
echo ║  运行以下命令启动服务：                                ║
echo ║  cd backend                                            ║
echo ║  npm start                                             ║
echo ║                                                        ║
echo ║  访问地址：http://localhost:3000                       ║
echo ║  登录账号：admin / 123456                              ║
echo ╚════════════════════════════════════════════════════════╝
echo.
pause
```

**2. 创建 start.bat（启动脚本）**

文件位置：`D:\桂在安康\start.bat`

内容：
```batch
@echo off
chcp 65001 >nul
echo ╔════════════════════════════════════════════════════════╗
echo ║          🌿 桂在安康 - 启动服务                       ║
echo ╚════════════════════════════════════════════════════════╝
echo.

cd backend
call npm start

pause
```

## 📱 打包成 APP（可选）

### 使用 Electron 打包成桌面应用

**1. 安装 Electron**
```bash
cd D:\桂在安康
npm init -y
npm install electron --save-dev
```

**2. 创建 main.js**
```javascript
const { app, BrowserWindow } = require('electron');
const path = require('path');

function createWindow() {
  const win = new BrowserWindow({
    width: 1200,
    height: 800,
    webPreferences: {
      nodeIntegration: true
    }
  });

  win.loadFile('index.html');
}

app.whenReady().then(createWindow);
```

**3. 修改 package.json**
```json
{
  "main": "main.js",
  "scripts": {
    "start": "electron ."
  }
}
```

**4. 打包**
```bash
npm install electron-packager -g
electron-packager . "桂在安康" --platform=win32 --arch=x64
```

生成可执行文件：`桂在安康.exe`

## 📞 常见问题

### Q1: MySQL 安装失败？
**A:** 
- 确保系统是 Windows 10/11
- 以管理员身份运行安装程序
- 关闭杀毒软件再试

### Q2: npm install 很慢？
**A:**
使用淘宝镜像：
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

### Q3: 数据库初始化失败？
**A:**
- 确保 MySQL 服务正在运行
- 检查 .env 文件中的密码是否正确
- 确保 MySQL 版本是 8.0+

### Q4: 别人无法访问？
**A:**
- 检查防火墙是否开放 3000 端口
- 确保在同一局域网
- 检查 IP 地址是否正确

### Q5: 如何制作成真正的 APP？
**A:**
有两种方式：
1. **桌面应用** - 使用 Electron 打包（见上面）
2. **手机 APP** - 使用 Cordova 或 React Native（需要额外开发）
