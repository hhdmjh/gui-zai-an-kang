## 🔍 常见问题排查

### Q1: 点击"重置表单"没反应
**检查**：
- 浏览器控制台是否有错误（F12）
- JavaScript 是否启用

**解决**：
- 刷新页面重试
- 清除浏览器缓存

---

### Q2: 点击"AI 健康建议"按钮没反应
**检查**：
- 是否有健康数据
- 网络连接是否正常
- 后端服务是否运行

**解决**：
- 先录入至少 1 条数据
- 检查 server.js 是否运行
- 查看浏览器控制台错误

---

### Q3: AI 报告页面显示空白
**检查**：
- 后端服务是否运行
- 数据库连接是否正常
- 是否有健康数据

**解决**：
- 重启后端服务
- 检查 MySQL 是否运行
- 录入更多数据后重试

---

### Q4: 数据库表不存在
**检查**：
```bash
# 登录 MySQL
mysql -u root -p

# 查看表
USE gui_zai_an_kang;
SHOW TABLES;

# 如果没有 ai_health_reports 表，运行更新脚本
```

**解决**：
```bash
cd d:\桂在安康\backend\database
node update-db.js
```
